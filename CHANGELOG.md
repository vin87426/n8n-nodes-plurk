# Changelog

All notable changes to this project will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/).

## [0.1.2] - 2026-06-28

### Changed

- Excluded the TypeScript incremental build cache from the published package.
- Explicitly included `README.md` and `LICENSE` in package files.

## [0.1.1] - 2026-06-28

### Changed

- Bumped the package version for the next patch release.

### Documentation

- Added the changelog.

## [0.1.0] - 2026-06-28

### Added

- Added the initial `Plurk` n8n community node.
- Added `Plurk OAuth1 API` credentials for Plurk OAuth 1.0a requests.
- Added categorized Plurk API 2.0 resources and operations.
- Added support for Plurk two-legged OAuth endpoints where applicable.
- Added binary upload support for endpoints that accept files.
- Added documentation for installation, credential setup, and local development.
